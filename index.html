  • Root filesystem mounting function Test on NetBSD

    In NetBSD's later stage of bootup, Kernel finds device with root filesystem. this search is assisted with the help of config file of kernel. NetBSD has Testing suite called ATF (Automated testing framework) But this suite does not cover this area of function. We attempt to write test for root device and filesystem selection. Functions we need to test are setroot , it utilizes number of other functions for different scenarios which are documented in my documentation . We will use vnd(8) for device simulation in userspace.
